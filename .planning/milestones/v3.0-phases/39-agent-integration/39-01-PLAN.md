**3.5.1 Overview and Timing**
- Detection runs AFTER peak picking (Step 3) and BEFORE writing LSD (Step 4)
- Purpose: Generate data-driven constraints to reduce LSD search space
- Four commands available: `lucy detect hybridisation`, `lucy detect neighbours`, `lucy detect hhb`, `lucy analyze grouping`
- Run detection ONCE per compound (results don't change between iterations)

**3.5.2 Selective Detection Strategy**
Teach the agent to query ONLY specific shift ranges where detection adds value (Option B from research):

| Shift Range | Detection Command | What to Query | Expected Constraints |
|-------------|------------------|---------------|---------------------|
| 120-160 ppm | `lucy detect hybridisation <shift>` | Aromatic/alkene detection | MULT hybridisation = 2 for sp2 |
| 160-220 ppm | `lucy detect hybridisation <shift>` AND `lucy detect neighbours <shift>` | Carbonyl confirmation | MULT hybridisation = 2, PROP mandatory O |
| 50-90 ppm | `lucy detect neighbours <shift>` | C-O/C-N bond detection | PROP mandatory O or N neighbor |
| < 50 ppm | Skip detection | Aliphatic region rarely ambiguous | None |

Note: For ALL 13C shifts, run `lucy analyze grouping` once with all shifts (fast, no database query).

For HHB detection: Run `lucy detect hhb <formula>` once if formula has 2+ heteroatoms.

**3.5.3 CLI Syntax and Interpretation**

For each command, provide:
1. Exact CLI syntax with all relevant flags
2. How to interpret output (threshold meanings)
3. How to translate to LSD constraints

**Hybridisation detection:**
```bash
lucy detect hybridisation <shift_ppm> --format json
```
Interpretation:
- sp2 > 80% → Set MULT hybridisation to 2 (sp2): `MULT N C 2 H`
- sp3 > 80% → Set MULT hybridisation to 3 (sp3): `MULT N C 3 H`
- Mixed (40-60%) → Ambiguous, use NMR evidence (DEPT/HSQC) as tiebreaker
- If database returns no data → Fall back to chemical shift heuristics (120-160 ppm = likely sp2)

**Neighbourhood detection:**
```bash
lucy detect neighbours <shift_ppm> --format json
```
Interpretation:
- Forbidden element (<1% frequency) → Consider PROP exclusion: `PROP L_carbon 0 L_element`
- Mandatory element (>95% frequency) → Add PROP requirement: `PROP L_carbon 1 L_element`
- Typical (1-95%) → No constraint, let LSD explore
- IMPORTANT: Check molecular formula FIRST — if formula has no oxygen, ignore "O mandatory" result

**HHB detection:**
```bash
lucy detect hhb <formula> --format json
```
Interpretation:
- Bond pair forbidden (<1%) → Do NOT add hetero-hetero BOND (e.g., no O-O)
- Bond pair allowed (>1%) → Allow in LSD (no exclusion needed)
- Note: HHB results are GUIDELINES. Override if NMR evidence strongly suggests rare bond.

**Signal grouping:**
```bash
lucy analyze grouping "<comma_separated_shifts>" --format json
```
Interpretation:
- Grouped signals → Use parenthesized HMBC in LSD: `HMBC (5 6) 10` instead of `HMBC 5 10`
- No groups → Use standard HMBC syntax
- Document all groups in CASE-PROGRESS.md

**3.5.4 LSD Constraint Translation (Examples)**

Provide 4 concrete examples showing detection output → LSD syntax:

1. **Aromatic carbon** (hybridisation sp2 at 132.5 ppm):
```
; Detection: lucy detect hybridisation 132.5 → sp2 = 92%
MULT 5 C 2 1    ; sp2 CH (hybridisation 2, from detection)
```

2. **Carbonyl carbon** (hybridisation sp2 + mandatory O at 175.3 ppm):
```
; Detection: lucy detect hybridisation 175.3 → sp2 = 99%
; Detection: lucy detect neighbours 175.3 → O mandatory (98%)
MULT 10 C 2 0   ; sp2 quaternary (carbonyl)
MULT 14 O 2 0   ; sp2 oxygen (carbonyl)
BOND 10 14       ; C=O (supported by both shift and detection)
```

3. **C-O ether carbon** (mandatory O neighbor at 65.3 ppm):
```
; Detection: lucy detect neighbours 65.3 → O mandatory (97%)
MULT 8 C 3 2    ; sp3 CH2 (from DEPT)
ELEM L_oxygen O
LIST L_co 8
PROP L_co 1 L_oxygen  ; C8 must have at least 1 O neighbor
```

4. **Close signals** (grouping at 130.2/130.4 ppm):
```
; Detection: lucy analyze grouping "130.2,130.4" → Group: [130.2, 130.4]
MULT 5 C 2 1
MULT 6 C 2 1
; Use parenthesized HMBC for ambiguous assignment:
HMBC (5 6) 10   ; either C5 or C6 correlates to H10
```
