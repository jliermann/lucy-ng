In models.py, ADD the following classes (keep all existing HybridisationDistribution and HybridisationResult classes untouched):

1. Add `from enum import Enum` import at top.

2. `ConstraintType(str, Enum)` with values:
   - FORBIDDEN = "forbidden"  (< threshold frequency)
   - TYPICAL = "typical"      (between thresholds)
   - MANDATORY = "mandatory"  (> threshold frequency)

3. `ElementConstraint(BaseModel)`:
   - element: str  (e.g., "C", "O", "N", "S", "halogen")
   - frequency: float  (0.0 to 1.0)
   - constraint_type: ConstraintType

4. `NeighbourDistribution(BaseModel)`:
   - carbon: float = 0.0
   - oxygen: float = 0.0
   - nitrogen: float = 0.0
   - sulfur: float = 0.0
   - halogen: float = 0.0
   - Method `get_constraints(self, forbidden_threshold: float = 0.01, mandatory_threshold: float = 0.95) -> list[ElementConstraint]`:
     Iterates over (element_name, frequency) pairs, classifies each, returns list.
   - Property `forbidden_elements` -> list[str]: Returns elements with frequency below a default 1% (convenience property, calls get_constraints internally or checks each field < 0.01)
   - Property `mandatory_elements` -> list[str]: Returns elements with frequency above 95%

5. `NeighbourResult(BaseModel)`:
   - shift_ppm: float
   - window_ppm: float
   - radius: int
   - forbidden_threshold: float = 0.01
   - mandatory_threshold: float = 0.95
   - distribution: NeighbourDistribution
   - constraints: list[ElementConstraint] = []
   - total_observations: int = 0
   - unique_hose_codes: int = 0
   - has_data: bool = False
   - warning: str | None = None
   - Method `summary(self) -> str`:
     Multi-line human-readable output following the HybridisationResult.summary() pattern:
     ```
     Neighbourhood Detection: 170.5 ppm (window +/- 2.0 ppm, radius 3)
     Distribution: C=85.0%, O=98.0%, N=12.0%
     Forbidden: S (<1.0%), halogen (<1.0%)
     Mandatory: O (>95.0%)
     Based on 1,000 observations from 42 HOSE codes
     ```

In detector.py:
6. Add import of the new models: `from lucy_ng.detection.models import NeighbourDistribution, NeighbourResult, ElementConstraint`

7. Add method `detect_neighbours()` to StatisticalDetector:
   ```python
   def detect_neighbours(
       self,
       shift_ppm: float,
       radius: int = 3,
       window_ppm: float = 2.0,
       forbidden_threshold: float = 0.01,
       mandatory_threshold: float = 0.95,
   ) -> NeighbourResult:
   ```
   Implementation:
   - Query records via self._db.get_hose_stats_by_shift_window(shift_ppm, radius, window_ppm)
   - Aggregate: has_carbon = sum(r.has_carbon_neighbor for r in records), same for O, N, S, halogen
   - total_observations = sum(r.count for r in records)
   - If total_observations == 0: return NeighbourResult with has_data=False and warning about regeneration
   - Check if ALL neighbour counts are 0 but total_observations > 0: this means the database has data but no neighbour columns populated. Return has_data=False with warning "Neighbour columns are unpopulated. Database needs regeneration with v5 schema."
   - Compute frequencies: carbon_freq = has_carbon / total_observations, etc.
   - Create NeighbourDistribution with computed frequencies
   - Get constraints via distribution.get_constraints(forbidden_threshold, mandatory_threshold)
   - Return NeighbourResult with all fields populated

In __init__.py:
8. Add NeighbourResult to imports and __all__:
   ```python

   __all__ = ["StatisticalDetector", "HybridisationResult", "NeighbourResult"]
   ```
