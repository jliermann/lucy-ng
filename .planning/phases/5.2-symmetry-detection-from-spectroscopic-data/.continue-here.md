<context>
This phase addressed the LSD "Odd number of sp2 atoms" error that occurred with Ibuprofen.
The root cause: molecular symmetry causes 13 carbons to produce only 9 NMR signals.
Without symmetry handling, the generator added 4 "missing" carbons as sp2, creating odd count.

Solution: AI-oriented convenience tools that expose:
1. Hydrogen budget (missing H indicates equivalent protonated carbons)
2. HSQC intensity (equivalent atoms show ~2× intensity)
3. Combined summary with interpretation hints

The AI uses these tools to reason about symmetry and generate appropriate LSD input
with SYME commands for equivalent atoms.

Architecture follows user's vision: "AI steers CLI/MCP tools like a computer-proficient
organic spectroscopist" - avoid hard-coding domain logic that AI can handle flexibly.
</context>
