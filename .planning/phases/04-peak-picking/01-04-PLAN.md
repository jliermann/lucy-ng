# Phase 4: 2D Peak Picking — Plan 01

## Goal

Implement 2D peak picker for HSQC, HMBC, COSY, and NOESY spectra, returning `PeakList2D` objects with ppm-converted positions.

## Context

### Already Implemented (Phase 2.1)

- `SimplePeakPicker` — threshold-based 1D peak picking
- `SimplePeakPicker.pick_peaks_with_noise_estimation()` — SNR-based 1D picking
- `AdaptivePeakPicker` — adaptive min-distance based on FWHM
- Data models: `Peak1D`, `Peak2D`, `PeakList1D`, `PeakList2D`

### What This Phase Adds

- `PeakPicker2D` class for 2D spectra (HSQC, HMBC, COSY, NOESY)
- Uses nmrglue's `ng.peakpick.pick()` under the hood
- Converts peak indices to ppm using unit converters

### Key Findings from Discovery

**nmrglue 2D Peak Picking API:**
```python
result = ng.peakpick.pick(data, pthres=threshold, nthres=None, msep=(3, 3))
# Returns numpy recarray with columns:
# - Y_AXIS, X_AXIS: peak positions in indices
# - cID: cluster ID
# - Y_LW, X_LW: line widths in each dimension
# - VOL: peak volume
```

**Index to PPM conversion:**
```python
uc_f1 = ng.fileiobase.uc_from_udic(udic, dim=0)
uc_f2 = ng.fileiobase.uc_from_udic(udic, dim=1)
f1_ppm = uc_f1.ppm(y_index)
f2_ppm = uc_f2.ppm(x_index)
```

## Tasks

### Task 1: Create PeakPicker2D class

Create `src/lucy_ng/processing/peak_picker_2d.py`:

```python
class PeakPicker2D:
    """2D NMR peak picker using nmrglue."""

    @staticmethod
    def pick_peaks(
        spectrum: Spectrum2D,
        threshold: float = 0.05,
        min_separation: tuple[int, int] = (3, 3),
        detect_negative: bool = False,
    ) -> PeakList2D:
        """Pick peaks from a 2D spectrum.

        Args:
            spectrum: 2D NMR spectrum with data and ppm scales
            threshold: Minimum intensity as fraction of max (0-1)
            min_separation: Minimum separation (F1_points, F2_points)
            detect_negative: Also detect negative peaks (for HMBC artifacts)

        Returns:
            PeakList2D containing detected peaks
        """
```

Implementation:
- Use `ng.peakpick.pick()` with `pthres` for positive peaks
- Optionally use `nthres` for negative peaks
- Convert Y_AXIS/X_AXIS indices to F1/F2 ppm using unit converters
- Store intensity from spectrum data at peak location
- Return `PeakList2D` with experiment metadata

**Acceptance**: Function picks peaks from 2D spectrum and returns PeakList2D.

## Risks

- **HMBC artifacts**: HMBC often has artifacts/noise peaks. Mitigation: Option for stricter filtering.
