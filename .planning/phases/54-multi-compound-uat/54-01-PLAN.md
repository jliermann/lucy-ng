  <action>
Start the full 928K-compound fragment extraction. This is a long-running process (~4-8 hours on M1 Mac) with built-in checkpointing.

```bash
lucy fragment build data/reference/lucy-ng-derep.db --fresh
```

IMPORTANT: This command takes 4-8 hours to complete. Run it as a background process so the user can monitor progress independently. The command has built-in checkpointing, so if interrupted it can be resumed with:


After the extraction completes (or after a reasonable runtime to confirm it is making progress), verify with:

