# Phase 10: NMRXiv Dataset Fetching

## Objective

Enable lucy-ng to download all NMR spectra for a compound from NMRXiv by DOI or project identifier.

## Execution Context

```

```

## Context

NMRXiv is an open NMR data repository. Public projects can be downloaded without authentication. Data is organized as Project → Sample/Study → Dataset → Spectra files (typically Bruker format).

**API base**: `https://nmrxiv.org/api/v1/`
**Download pattern**: `https://nmrxiv.org/{username}/download/{project}/{study}/{filename}`

### Task 2: Implement API Fetching

Implement the actual API calls and download logic.

**Key implementation details:**

1. **Identifier resolution:**
   - DOI format: `10.57992/NMRXIV.P10.S69` → extract project `P10`, study `S69`
   - DOI (project only): `10.57992/NMRXIV.P10` → extract project `P10`
   - URL format: `https://nmrxiv.org/P10` → extract `P10`
   - Direct ID: `P10` or `S69` → use as-is

   When DOI specifies a study, download that study by default.
   Use `--all` flag to download entire project.

2. **API calls:**
   - `GET /api/v1/P7` → project metadata
   - Parse response to get studies list
   - For each study, get datasets and files

3. **Download:**
   - Use discovered download URL pattern
   - Download files preserving directory structure
   - Show progress with tqdm

### Task 3: Add CLI Command

Add `lucy fetch nmrxiv` command.

**File:** `src/lucy_ng/cli/fetch.py` (new)

**Command structure:**
```bash
lucy fetch nmrxiv IDENTIFIER [OPTIONS]

Arguments:
  IDENTIFIER  NMRXiv DOI or ID (e.g., 10.57992/NMRXIV.P10.S69 or P10)

Options:
  -o, --output PATH     Output directory (default: current directory)
  --all                 Download entire project (when DOI specifies a study)
  --format [text|json]  Output format
```

**Example usage:**
```bash
# By DOI (downloads specific study S69)
lucy fetch nmrxiv 10.57992/NMRXIV.P10.S69 --output ./data/

# By DOI with --all (downloads entire project P10)
lucy fetch nmrxiv 10.57992/NMRXIV.P10.S69 --all --output ./data/

# By project ID (downloads all studies)
lucy fetch nmrxiv P10 --output ./data/

# JSON output for scripting
lucy fetch nmrxiv P10 --format json
```
