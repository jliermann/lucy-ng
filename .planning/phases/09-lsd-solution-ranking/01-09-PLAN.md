**Local Predictor Advantages:**
- Fast: <100ms per molecule (vs ~1-2s for REST API)
- Offline: No network dependency
- Better coverage: COCONUT has ~895K molecules (vs ~33K in nmrshiftdb2)
- Rich metadata: Confidence scores, radius used, match counts
</execution_context>

<context>
**From Phase 8 Implementation:**
- `C13Predictor.predict_from_smiles(smiles)` returns `PredictionResult`
- `PredictionResult.predictions` is list of `PredictedShift` with `shift`, `confidence`, etc.
- Predictor uses 6→1 radius fallback for maximum coverage
- Requires pre-built lookup table (HOSE→shift mappings from COCONUT)

**Critical Pitfalls:**
1. Atom numbering: SMILES atom order vs experimental peaks — use greedy shift matching
2. Equivalent carbons: Multiple atoms may map to same experimental peak
3. Missing predictions: Some atoms may not have HOSE code matches — use confidence weighting

**Integration Pattern:**
- Input: List of `LSDSolution` objects + experimental 13C peaks (list of ppm values) + lookup table path
- Output: `RankedSolution` list sorted by score (best first)

**Scoring Strategy:**
- Primary: MAE (Mean Absolute Error) of matched shifts — lower is better
- Secondary: Penalize unmatched peaks
- Tertiary: Weight by prediction confidence (optional)
</context>

## Task 1: Create Ranking Model

**File:** `src/lucy_ng/ranking/models.py`

Create Pydantic v2 model for ranked solutions. Reuse `PredictionResult` from `lucy_ng.prediction`.

```python
from pydantic import BaseModel, Field, computed_field
from lucy_ng.lsd.parser import LSDSolution
from lucy_ng.prediction.models import PredictionResult


class ShiftAssignment(BaseModel):
    """Assignment between predicted and experimental shift."""
    predicted_idx: int = Field(description="Index in predicted shifts list")
    experimental_idx: int = Field(description="Index in experimental shifts list")
    predicted_ppm: float = Field(description="Predicted shift value")
    experimental_ppm: float = Field(description="Experimental shift value")
    error: float = Field(description="Absolute error |predicted - experimental|")


class RankedSolution(BaseModel):
    """LSD solution with ranking score based on 13C prediction."""
    solution: LSDSolution
    prediction: PredictionResult
    experimental_shifts: list[float] = Field(description="Experimental 13C peaks (ppm)")
    assignments: list[ShiftAssignment] = Field(description="Matched shift pairs")
    mae: float = Field(description="Mean Absolute Error of matched shifts")
    matched_count: int = Field(description="Number of matched peaks")
    unmatched_predicted: int = Field(description="Predicted shifts without match")
    unmatched_experimental: int = Field(description="Experimental peaks without match")
    avg_confidence: float = Field(description="Average confidence of predictions")

    @computed_field
    @property
    def score(self) -> float:
        """Combined score (lower is better). MAE + penalty for unmatched."""
        penalty = (self.unmatched_predicted + self.unmatched_experimental) * 5.0
        return self.mae + penalty
```
