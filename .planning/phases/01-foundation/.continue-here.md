<context>
lucy-ng is an AI-agent powered CASE (Computer-Assisted Structure Elucidation) tool for organic natural products. It's the spiritual successor to Lucy (sold to Bruker). The key differentiator is programmatic/unattended operation - Claude can iterate through the elucidation process without human intervention.

The architecture is: Python library (core) → CLI (human testing) → MCP server (Claude agent). Input is Bruker NMR data (1D: 1H, 13C; 2D: HSQC, HMBC), output is LSD/pyLSD input files and parsed solutions.

Config is YOLO mode (auto-approve) with standard depth (5-8 phases).
</context>
