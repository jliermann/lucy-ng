<objective>
Confirm empirically that the LSD binary (LSD-3.4.9) silently ignores a `FORM` line in an LSD input file, and document this as a reproducible-research audit trail per D-15.

Purpose: Phase 66 added `emit_form()` to LSDInputGenerator so pyLSD-format files include a `FORM` declaration. Before shipping Phase 69, we must confirm the LSD binary used in CI/dev tolerates this — otherwise the Phase 66 change would silently break `lucy lsd run` for files generated in `pylsd_mode`. D-15 requires both a living pytest test and a findings document.

Output: test_lsd_form_tolerance.py (skipif LSD missing), two minimal fixture files, .planning/findings/form-tolerance.md.
</objective>
