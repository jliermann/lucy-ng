Run a minimal end-to-end test of the Agent Teams API. This test validates the full lifecycle WITHOUT running actual CASE work.

**Pre-check:**
```bash
echo $CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS
```
If not "1", set it: `export CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1`

**Test sequence:**

1. **Create team:**
   ```
   TeamCreate(
     team_name="case-validation-test",
     description="Phase 41 API validation — temporary test team"
   )
   ```
   Verify: ~/.claude/teams/case-validation-test/ exists

2. **Create a test task:**
   ```
   TaskCreate(
     subject="validation-echo",
     description="Reply to team lead with a confirmation message. Then mark this task completed.",
     activeForm="Validating team communication"
   )
   ```

3. **Spawn ONE test teammate** (use nmr-chemist stub — simplest test):
   ```
   Task(
     name="test-agent",
     team_name="case-validation-test",
     subagent_type="lucy-nmr-chemist",
     model="sonnet",
     prompt="You are a test agent for API validation. Do the following:
            1. Call TaskList to see available tasks
            2. Claim the validation-echo task via TaskUpdate
            3. Send a message to team lead: SendMessage(type='message', recipient='team-lead', content='Validation successful — I can see tasks and send messages.', summary='Validation OK')
            4. Mark the task completed via TaskUpdate
            Then wait for shutdown request."
   )
   ```
   Use model="sonnet" (cheaper for validation, opus not needed).

4. **Wait for response message from test-agent.** The message should arrive automatically.

5. **Verify team state:**
   ```bash
   cat ~/.claude/teams/case-validation-test/config.json
   ```
   Verify: members array contains test-agent entry.

   ```
   TaskList()
   ```
   Verify: validation-echo task shows status "completed" with owner "test-agent".

6. **Test SendMessage from lead to teammate:**
   ```
   SendMessage(
     type="message",
     recipient="test-agent",
     content="Team lead confirms: bidirectional messaging works.",
     summary="Bidirectional messaging test"
   )
   ```

7. **Shutdown teammate:**
   ```
   SendMessage(
     type="shutdown_request",
     recipient="test-agent",
     content="Validation complete. Please shut down."
   )
   ```
   Wait for shutdown confirmation.

8. **Delete team:**
   ```
   TeamDelete()
   ```
   Verify: ~/.claude/teams/case-validation-test/ no longer exists.

**Record results** — document what worked and what didn't. If any step fails, note the error message and conditions.

**Important:** This is a SMOKE TEST, not a CASE run. No NMR data, no LSD solver, no compound directory needed. The goal is purely to validate that TeamCreate/Task(team_name)/SendMessage/TeamDelete work in a skill context.
