  <files></files>
  <action>
  Run detection CLI commands against the regenerated database and record results. This validates that the full pipeline (SDF -> HOSE generation -> aggregation -> database -> query -> CLI output) produces scientifically correct answers.

  **Hybridisation detection validation:**

  Run these commands and record output:
  ```bash
  # Aromatic region - should show sp2 dominant
  lucy detect hybridisation 128.0 --format json
  lucy detect hybridisation 135.0 --format json

  # Aliphatic region - should show sp3 dominant
  lucy detect hybridisation 25.0 --format json
  lucy detect hybridisation 40.0 --format json

  # Carbonyl region - should show sp2 dominant
  lucy detect hybridisation 175.0 --format json
  lucy detect hybridisation 200.0 --format json

  # Alkyne region - should show sp1 present
  lucy detect hybridisation 75.0 --format json
  ```

  **Neighbour detection validation:**

  Run these commands:
  ```bash
  # Carbonyl region - oxygen should be mandatory/common
  lucy detect neighbours 175.0 --format json
  lucy detect neighbours 200.0 --format json

  # Aromatic region - carbon mandatory (ring)
  lucy detect neighbours 128.0 --format json

  # Aliphatic region - carbon common, others variable
  lucy detect neighbours 25.0 --format json
  ```

  **HHB detection validation:**

  Run these commands:
  ```bash
  # Ibuprofen formula - should have C-O bond data
  lucy detect hhb C13H18O2 --format json

  # Pulegone formula - should have C-O bond data
  lucy detect hhb C10H16O --format json

  # Simple hydrocarbon - should have no heteroatom bonds
  lucy detect hhb C10H14 --format json
  ```

  **Ibuprofen-specific detection:**

  Using known ibuprofen 13C shifts (from CASE1 analysis), run detection on key carbons:
  ```bash
  # Aromatic carbons: 127.3, 129.4, 137.2, 140.9
  lucy detect hybridisation 127.3 --format json  # sp2
  lucy detect hybridisation 140.9 --format json  # sp2

  # Carboxylic acid: 181.1
  lucy detect hybridisation 181.1 --format json  # sp2
  lucy detect neighbours 181.1 --format json     # O mandatory

  # Aliphatic CH: 44.9, 45.0
  lucy detect hybridisation 44.9 --format json   # sp3
  ```

  Record all results for the validation report. Note any unexpected or incorrect frequencies.
  </action>
  <verify>
  - Aromatic shifts return sp2 frequency > 0.80
  - Aliphatic shifts return sp3 frequency > 0.80
  - Carbonyl shifts return oxygen as common/mandatory neighbour
  - Ibuprofen-specific checks return chemically correct results
  </verify>
