      def detect_intensity_symmetry(
          peaks: list[Peak1D],
          aromatic_ch_ppms: list[float],
          tolerance_ppm: float = 1.0,
          min_ratio: float = 1.6,
      ) -> list[tuple[float, float, int]]

    Algorithm (reconciled with the interface block above):
    1. If len(aromatic_ch_ppms) < 2: return []
    2. Collect all_aromatic = all peaks from the peaks list where 100.0 <= p.position <= 165.0 (this is the reference class for the median computation, providing the 1C baseline from Cq peaks)
    3. If len(all_aromatic) < 2: return []
    4. Compute median_intensity = float(np.median([p.intensity for p in all_aromatic]))
    5. If median_intensity <= 0: return []
    6. Filter candidate_peaks = peaks where ANY abs(p.position - ref) < tolerance_ppm for ref in aromatic_ch_ppms AND 100.0 <= p.position <= 165.0 (scope restriction D-06)
    7. Return list of (p.position, p.intensity / median_intensity, round(p.intensity / median_intensity)) for each candidate_peak where p.intensity / median_intensity >= min_ratio
    8. Sort results by ppm descending

    The function does NOT modify detect_aromatic_cosy_pairs — it only provides input data to the agent, which then calls lucy detect aromatic-cosy. DO NOT touch generator.py.
