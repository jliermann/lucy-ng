<!-- Key background: why these 3 correlations are 4J -->
<!-- From v4.0 UAT post-mortem (MEMORY.md):                                    -->
<!-- Ibuprofen is a para-disubstituted benzene: CC(C)Cc1ccc(cc1)C(C)C(=O)O    -->
<!-- The W-pathway enables strong 4J HMBC couplings between:                   -->
<!--   ArCH at ~129 ppm (ortho to isobutyl side) ↔ CH2 at ~45 ppm (benzylic)  -->
<!--   ArCH at ~127 ppm (ortho to COOH side)    ↔ CH  at ~45 ppm (alpha COOH) -->
<!-- These look like valid 3J HMBC couplings, but the actual bond path is 4J.  -->
<!-- LSD enforces HMBC as 2-3J, so it accepts these as constraints BUT then    -->
<!-- cannot form the aromatic ring because the bond distance is wrong.          -->
<!-- Consequence: all v4.0 solutions have wrong ring sizes (5-9 membered),     -->
<!-- zero aromatic atoms, despite 5 sp2 carbons at 127-141 ppm.                -->
<!--                                                                            -->
<!-- The ibuprofen.lsd in project root uses this atom numbering:               -->
<!--   Atom 4:  C 140.84 ppm  Aromatic quaternary (C bearing isobutyl side)    -->
<!--   Atom 5:  C 136.96 ppm  Aromatic quaternary (C bearing COOH side)        -->
<!--   Atoms 6,7: C 129.38 ppm  Aromatic CH (ortho to isobutyl)                -->
<!--   Atoms 8,9: C 127.26 ppm  Aromatic CH (ortho to COOH)                    -->
<!--   Atom 10: C  45.03 ppm  CH aliphatic (alpha to COOH)                     -->
<!--   Atom 11: C  44.90 ppm  CH2 aliphatic (benzylic)                         -->
<!--                                                                            -->
<!-- The 4J W-pathway correlations in ibuprofen.lsd are:                       -->
<!--   HMBC 6 8  - ArCH 129.38 → H on ArCH 127.26 (4J through ring, W-path)   -->
<!--   HMBC 10 6 - CH 45.03 → H on ArCH 129.38 (4J benzylic W-path)           -->

    Run RDKit aromatic ring check on each SMILES. Use this inline Python snippet (no new file needed — run directly with python3 -c or as a heredoc):
    ```python
    from rdkit import Chem

    with open(".planning/phases/65-hypothesis-gate/ibuprofen_no4j.smi") as f:
        lines = [l.strip() for l in f if l.strip()]

    results = []
    for i, line in enumerate(lines):
        parts = line.split()
        smiles = parts[0] if parts else ""
        mol = Chem.MolFromSmiles(smiles)
        if mol is None:
            results.append({"rank": i+1, "smiles": smiles, "aromatic_atoms": 0, "valid": False})
            continue
        aromatic_atoms = sum(1 for a in mol.GetAtoms() if a.GetIsAromatic())
        results.append({"rank": i+1, "smiles": smiles, "aromatic_atoms": aromatic_atoms, "valid": True})
        print(f"Rank {i+1}: {smiles} | aromatic_atoms={aromatic_atoms}")

    aromatic_solutions = [r for r in results if r["aromatic_atoms"] > 0]
    print(f"\nTotal solutions: {len(results)}")
    print(f"Solutions with aromatic ring: {len(aromatic_solutions)}")
    if aromatic_solutions:
        print("HYPOTHESIS CONFIRMED: aromatic ring solutions present")
        print(f"First aromatic solution at rank {aromatic_solutions[0]['rank']}: {aromatic_solutions[0]['smiles']}")
    else:
        print("HYPOTHESIS FAILED: no aromatic ring solutions")
    ```
