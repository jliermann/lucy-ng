### Badlist Filters (Exclude Strained Rings)

Natural products almost never contain 3- or 4-membered rings. Add these exclusions to EVERY LSD file, placed after all MULT/HSQC/HMBC/BOND lines and before any ELIM:

```
; Exclude strained rings
DEFF NOT C1CC1      ; no cyclopropane
DEFF NOT C1CCC1     ; no cyclobutane
DEFF NOT C1NC1      ; no aziridine
DEFF NOT C1NCC1     ; no azetidine
DEFF NOT C1SC1      ; no thiirane
DEFF NOT C1SCC1     ; no thietane
DEFF NOT C1OC1      ; no epoxide (remove this line if strong epoxide evidence)
DEFF NOT C1OCC1     ; no oxetane (remove this line if strong oxetane evidence)
```

**Exception:** If the 13C spectrum shows shifts in the 45-55 ppm range AND the molecular formula contains oxygen, the compound may have an epoxide. In that case, remove the `DEFF NOT C1OC1` line to allow epoxide-containing solutions.
```

Also update the "Manual File Construction Checklist" (around line 289-297) to add item 8:

```

**Addition 2: Ranking interpretation update**

Find the section that discusses ranking/solution ranking (near section 8 or wherever ranking is discussed). Add or update to include:

```markdown
### Ranking Algorithm (Two-Tier)

`lucy lsd rank` ranks solutions by:
1. **Match count** (descending) — how many predicted carbons have an experimental shift within tolerance
2. **MAE** (ascending) — mean absolute error among all predicted shifts

A solution with 13/13 matched signals and MAE 2.5 is BETTER than a solution with 11/13 matches and MAE 1.8 (incomplete coverage suggests wrong structure). Always report BOTH metrics when discussing ranking results.
```
