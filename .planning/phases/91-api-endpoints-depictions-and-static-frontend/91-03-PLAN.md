<objective>
Deliver the candidate-structures widget: `GET /api/structures` (list with rank/mae/quality
or unranked fallback, WV-04) and `GET /api/structure/{i}.svg` (on-demand RDKit SVG,
WV-04) with graceful degradation (WV-06). SVG rendering lives in a standalone
`depiction.py`; the structures router imports it lazily inside `make_router()` so RDKit
never loads at webview package import time (WV-08 import-safety).

Purpose: this is the visually central widget — the top candidate structures the user
watches emerge during a run. During a live run only `iteration_NN/solutions.smi` exists,
so the unranked fallback is the primary path, not an edge case.
Output: src/lucy_ng/webview/depiction.py, src/lucy_ng/webview/routers/structures.py.
</objective>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| localhost client → SVG endpoint | The path parameter `i` crosses from client to handler; must be a bounded integer index, never a filesystem path |
| filesystem → depiction | Untrusted SMILES strings from solutions.smi / ranking_results.json are rendered to SVG |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-91-05 | Tampering / path traversal | GET /api/structure/{i}.svg | mitigate | `i` is FastAPI-typed `int` (non-integer/`../` rejected as 422 before the handler); no path component is built from `i` — it indexes an in-memory list only |
| T-91-06 | Information Disclosure | Out-of-range index reveals list size | mitigate | Return 404 with a generic detail string; do not echo len(structures) or array bounds |
| T-91-07 | Spoofing / XSS | Malicious SMILES injecting script into SVG | mitigate | RDKit emits only drawing primitives (path/circle/text); SMILES cannot inject `<script>` into RDKit-generated SVG; placeholder_svg is a static string with no script (verified via grep gate in Task 1) |
| T-91-08 | Denial of Service | Pathological SMILES slow to render | accept | Localhost-only single-user monitor; RDKit render of drug-like molecules is sub-millisecond; no external attacker. LOW per RESEARCH §Security Domain |
| T-91-SC | Tampering | package installs | accept | No new packages; RDKit is an existing core dependency |
</threat_model>
