Response payload shapes (Claude's discretion, locked here so tests and impl agree):

GET /api/status  → JSON object:
  { "state": "waiting"|"running"|"complete",
    "iteration": int|null, "active_phase": str|null, "elapsed_s": int|null }

GET /api/structures → JSON object:
  { "source": "waiting"|"unranked"|"ranked", "total": int,
    "structures": [ { "index": int, "smiles": str,
                      "rank": int|null, "mae": float|null, "quality": str|null } ] }

GET /api/structure/{i}.svg → image/svg+xml body; 404 when i out of range.
  Index i is 0-based (matches array position in /api/structures.structures).

GET /api/log → JSON object: { "state": "ok"|"waiting", "content": str }

GET / → text/html (the static index.html)

Router factory contract (each router module in lucy_ng.webview.routers):
  make_router(analysis_dir: Path) -> fastapi.APIRouter   # routes use prefix "/api"

Depiction contract (lucy_ng.webview.depiction):
  render_smiles(smiles: str, width: int = 300, height: int = 300) -> str | None
  placeholder_svg(width: int = 300, height: int = 300) -> str
</interfaces>
