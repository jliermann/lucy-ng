Add an `advisory` field to `NeighbourResult` so that every `lucy detect neighbours` output — in both JSON and text format — self-labels as a statistical prior that must not be encoded as a hard LSD constraint without direct connectivity evidence. Purpose: The advisory label makes the constraint encoded in Plan 01's skill edits machine-verifiable: agents consuming JSON output see `"advisory": true` and the note string directly in the output object, making it harder to silently treat the result as a hard constraint.

Output: Two modified source files and one updated test file. All three quality gates (pytest, ruff, mypy) pass.
