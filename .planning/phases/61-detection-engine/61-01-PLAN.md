Add `detect_4j_coupling` method to StatisticalDetector in detector.py:

```
def detect_4j_coupling(
    self,
    carbon_shift: float,
    h_carbon_shift: float,
    *,
    window_ppm: float = 2.0,
    likely_threshold: float = 0.50,
    possible_threshold: float = 0.15,
    min_observations: int = 50,
) -> CouplingPathResult:
```

Algorithm:
1. Get HOSE codes matching carbon_shift via `get_hose_stats_by_shift_window(carbon_shift, radius=2, window_ppm)` — extract unique hose_code values
2. Get HOSE codes matching h_carbon_shift similarly
3. For each (carbon_hose, h_hose) pair, call `get_coupling_path_stats(carbon_hose, h_hose)` — collect all records
4. If no records found (exact pair miss), try fallback: for each carbon_hose, call `get_coupling_path_stats_by_carbon(carbon_hose)` — aggregate all partners. Set used_fallback=True.
5. If still no records, return CouplingPathResult with has_data=False
6. Aggregate: sum counts by bond_distance across all collected records. Compute total. If total < min_observations, return risk_level=insufficient_data
7. Compute probabilities: j2 = count_2/total, j3 = count_3/total, j4 = count_4/total, j5_plus = count_5plus/total
8. Classify: p_long_range = j4 + j5_plus. If >= likely_threshold: likely_4j + "defer". If >= possible_threshold: possible_4j + "HMBC X Y 2 4". Else: unlikely_4j + "normal".
