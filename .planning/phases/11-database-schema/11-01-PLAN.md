Create DatabaseManager class:

```python
class DatabaseManager:
    def __init__(self, db_path: str | Path):
        """Initialize with path to SQLite database."""

    def create_tables(self) -> None:
        """Create tables if they don't exist."""

    def get_by_formula(self, formula: str) -> list[CompoundRecord]:
        """Get all compounds matching normalized formula."""

    def insert_compound(self, compound: CompoundRecord, shifts: list[ShiftRecord]) -> int:
        """Insert compound and its shifts, return compound ID."""

    def insert_compounds_batch(self, compounds: list[tuple[CompoundRecord, list[ShiftRecord]]]) -> int:
        """Batch insert for performance. Return count inserted."""

    def get_compound_count(self) -> int:
        """Return total compounds in database."""

    def get_formula_count(self) -> int:
        """Return count of unique formulas."""

    def close(self) -> None:
        """Close database connection."""
```

Key implementation details:
- Use connection pooling with check_same_thread=False for thread safety
- Batch inserts use executemany() with transactions (commit every 1000)
- get_by_formula normalizes input formula before query
- Return CompoundRecord with shifts populated via JOIN query
- Use context manager protocol (__enter__/__exit__) for safe cleanup
