# Phase 4.1: 2D Peak Picking Validation — Plan 01

## Goal

Implement cross-validation between HSQC peaks and 1D 13C peaks to ensure 2D peak picking produces scientifically plausible results.

## Context

### Physical Constraint

Every HSQC peak represents a direct C-H correlation. Therefore, the F1 (13C) position of each HSQC peak MUST correspond to a peak in the 1D 13C spectrum. If an HSQC peak has no corresponding 1D 13C peak, it's either:
1. An artifact/noise peak in the HSQC
2. A peak missed by the 1D peak picker (threshold too high)

### Discovery Findings

Testing with Ibuprofen data (HSQC exp 6, 13C exp 2):

| HSQC F1 (ppm) | 1D 13C Match | Delta |
|---------------|--------------|-------|
| 129.4 | 129.38 | 0.02 |
| 127.3 | 127.26 | 0.04 |
| 77.2 | 77.00 | 0.20 |
| 45.0-45.1 | 45.03 | 0.03-0.07 |
| 22.4 | 22.37 | 0.03 |
| 18.1 | 18.09 | 0.01 |
| 130.8, 126.0, 125.8... | NO MATCH | — |

Several HSQC peaks have no 1D match, suggesting either:
- HSQC threshold too low (picking noise)
- 1D threshold too high (missing real peaks)

### Expected Behavior

For a valid HSQC peak list:
- High match rate (>80%) indicates reliable peak picking
- Low match rate suggests threshold adjustment needed
- Unmatched peaks should be flagged for review

## Tasks

### Task 1: Create PeakValidator class

Create `src/lucy_ng/processing/peak_validator.py`:

```python
@dataclass
class ValidationResult:
    """Result of peak list validation."""
    matched_peaks: list[tuple[Peak2D, Peak1D]]
    unmatched_2d_peaks: list[Peak2D]
    unmatched_1d_peaks: list[Peak1D]
    match_rate: float
    tolerance_used: float

class PeakValidator:
    """Validates 2D peaks against 1D reference peaks."""

    @staticmethod
    def validate_hsqc_against_1d(
        hsqc_peaks: PeakList2D,
        carbon_peaks: PeakList1D,
        tolerance: float = 0.5,
    ) -> ValidationResult:
        """Validate HSQC peaks against 1D 13C peaks."""
```

Implementation:
- For each HSQC peak, find nearest 1D 13C peak within tolerance
- Track matched pairs, unmatched HSQC peaks, unmatched 1D peaks
- Calculate match rate
- Return structured ValidationResult

**Acceptance**: Function validates HSQC against 1D 13C and returns ValidationResult.
