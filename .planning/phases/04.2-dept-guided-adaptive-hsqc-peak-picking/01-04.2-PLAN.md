# Phase 4.2: DEPT-Guided Adaptive HSQC Peak Picking — Plan

## Objective

Create an adaptive peak picker that uses DEPT-135 data as ground truth to tune HSQC peak picking thresholds, ensuring all protonated carbons are detected while filtering noise.

## Context

### Problem Statement
- Fixed HSQC threshold (0.05) misses weak signals (e.g., 30.15 ppm in Ibuprofen)
- Lowering threshold (0.02) finds all carbons but picks 113 peaks (69 unique) vs 7 real protonated carbons
- Need adaptive approach: lower threshold until all DEPT carbons matched, then filter

### Key Insight
DEPT-135 provides **ground truth** for protonated carbons:
- CH and CH3 appear as positive peaks
- CH2 appears as negative peaks
- Quaternary carbons are silent

Every carbon in DEPT-135 MUST have a corresponding HSQC signal.

### Existing Components
- `SimplePeakPicker` — 1D peak picking (used for DEPT)
- `PeakPicker2D` — 2D peak picking with threshold parameter
- `PeakValidator` — validates 2D peaks against 1D reference
- `AdaptivePeakPicker` — exists for 1D (FWHM-based), provides pattern for adaptive approach

### Task 1: Create DEPTGuidedPicker class

Create `src/lucy_ng/processing/dept_guided_picker.py`:

```python
class DEPTGuidedPicker:
    """DEPT-guided adaptive HSQC peak picker.

    Uses DEPT-135 peaks as ground truth to adaptively tune HSQC
    peak picking threshold until all protonated carbons are found.
    """

    @staticmethod
    def pick_hsqc_peaks(
        hsqc: Spectrum2D,
        dept135: Spectrum1D,
        dept_threshold: float = 0.02,
        initial_hsqc_threshold: float = 0.10,
        min_hsqc_threshold: float = 0.005,
        tolerance: float = 1.0,
    ) -> DEPTGuidedResult:
        """Pick HSQC peaks guided by DEPT-135 ground truth.

        Algorithm:
        1. Pick DEPT-135 peaks to get ground truth protonated carbons
        2. Start with high HSQC threshold
        3. Lower threshold iteratively until all DEPT carbons matched
        4. Filter HSQC peaks to only those matching DEPT positions
        5. Return validated, noise-free peak list
        """
```

Result dataclass:
```python
@dataclass
class DEPTGuidedResult:
    peaks: PeakList2D  # Validated HSQC peaks
    dept_peaks: PeakList1D  # DEPT reference peaks

    iterations: int  # Number of threshold adjustments
    all_carbons_found: bool  # Whether all DEPT carbons were matched
    unmatched_dept_peaks: list[Peak1D]  # DEPT peaks not found in HSQC
```

### Task 2: Implement adaptive threshold algorithm

Core algorithm in `DEPTGuidedPicker.pick_hsqc_peaks()`:

```python
# 1. Pick DEPT-135 peaks
dept_peaks = SimplePeakPicker.pick_peaks(dept135, threshold=dept_threshold)
dept_positions = [p.position for p in dept_peaks.peaks]

# 2. Iteratively lower HSQC threshold
threshold = initial_hsqc_threshold
iterations = 0
threshold_step = 0.5  # Multiplicative reduction

while threshold >= min_hsqc_threshold:
    iterations += 1

    # Pick HSQC peaks at current threshold
    hsqc_peaks = PeakPicker2D.pick_peaks(hsqc, threshold=threshold)
    hsqc_carbons = set(p.f1_position for p in hsqc_peaks.peaks)

    # Check if all DEPT carbons are matched
    unmatched = []
    for dept_ppm in dept_positions:
        if not any(abs(c - dept_ppm) <= tolerance for c in hsqc_carbons):
            unmatched.append(dept_ppm)

    if len(unmatched) == 0:
        break  # All carbons found

    threshold *= threshold_step  # Lower threshold

# 3. Filter HSQC peaks to only validated ones
validated_peaks = PeakValidator.filter_validated_peaks(
    hsqc_peaks, dept_peaks, tolerance=tolerance
)
```

**Tests to write:**
