# Phase 4.2: DEPT-Guided Adaptive HSQC Peak Picking — Summary

## Outcome

Successfully implemented DEPT-guided adaptive peak picking that uses DEPT-135 as ground truth to tune HSQC thresholds, ensuring all protonated carbons are detected while filtering noise.

## What Was Built

### DEPTGuidedPicker Class

- `pick_hsqc_peaks()` — Adaptive threshold algorithm guided by DEPT-135
- `pick_hsqc_peaks_with_dept90()` — Enhanced version using DEPT-90 for CH/CH3 distinction
- `_pick_dept_peaks()` — Internal method to pick DEPT peaks including negative CH2
- `_extract_multiplicities()` — Extract CH/CH3 vs CH2 from DEPT sign

### DEPTGuidedResult Dataclass

- `peaks` — Validated HSQC peaks (filtered to match DEPT carbons)
- `dept_peaks` — DEPT-135 reference peaks used as ground truth
- `threshold_used` — Final HSQC threshold that found all carbons
- `iterations` — Number of threshold adjustments made
- `all_carbons_found` — Whether all DEPT carbons were matched
- `unmatched_dept_peaks` — DEPT peaks not found (if threshold floor reached)
- `carbon_multiplicities` — Map of ppm → multiplicity ("CH", "CH2", "CH3", "CH/CH3")
- `summary()` — Human-readable result summary

### Algorithm

1. Pick DEPT-135 peaks → ground truth protonated carbons
2. Start HSQC at initial threshold (default 0.10)
3. Lower threshold iteratively (×0.5) until all DEPT carbons matched
4. Filter HSQC peaks to only those matching DEPT positions
5. Extract multiplicity from DEPT sign (positive=CH/CH3, negative=CH2)
6. Optionally refine with DEPT-90 (CH visible, CH3 not)

## Key Technical Decisions

| Decision | Rationale |
|----------|-----------|
| Multiplicative threshold reduction | 0.5 factor gives logarithmic steps (0.10 → 0.05 → 0.025...) |
| DEPT-135 for ground truth | Only DEPT-135 shows both positive and negative peaks |
| Tolerance 1.0 ppm default | Accounts for slight shift differences between 1D and 2D |
| AdaptivePeakPicker for DEPT | Supports negative peak detection for CH2 |

## Usage Example

```python
from lucy_ng import BrukerReader, DEPTGuidedPicker

# Load spectra
hsqc = BrukerReader.read_2d("data/Ibuprofen/6")
dept135 = BrukerReader.read_1d("data/Ibuprofen/3")

# Run DEPT-guided peak picking
result = DEPTGuidedPicker.pick_hsqc_peaks(hsqc, dept135)

print(result.summary())
# All carbons found, noise filtered, multiplicities extracted

# Optional: refine with DEPT-90 for CH vs CH3
dept90 = BrukerReader.read_1d("data/Ibuprofen/4")
result = DEPTGuidedPicker.pick_hsqc_peaks_with_dept90(hsqc, dept135, dept90)
```
