<objective>
Implement SolutionMerger: InChI-key deduplication of solutions from multiple LSD permutation runs, with provenance tracking and JSON reporting. Wire exports into the lsd package.

Purpose: After the orchestrator runs LSD multiple times with different correlation configurations, the merger combines all solutions into a single deduplicated set. Each unique structure is tracked back to which permutation(s) produced it, enabling the downstream agent to understand which 4J correlation configuration found the correct answer.

Output: `SolutionMerger`, `MergedSolution`, `MergeResult` classes added to `orchestrator.py`. Package exports updated in `__init__.py`. Tests appended to `test_lsd_orchestrator.py`.
</objective>
