# Phase 3: 2D NMR Reading — Plan 01

## Goal

Implement a Bruker 2D file reader that loads processed 2D NMR data (HSQC, HMBC, COSY) and returns populated `Spectrum2D` objects.

## Context

- nmrglue library used for 1D reading (Phase 2)
- `Spectrum2D` model already exists with fields: data, f1_ppm_scale, f2_ppm_scale, f1_nucleus, f2_nucleus, experiment_type, frequency, metadata
- Test data available: Ibuprofen HSQC (exp 6), HMBC (exp 7), COSY (exp 5)
- Additional test data: 4-Hydroxy-3-Iodo-biphenyl, PSP, Sali_Eth datasets

### Key Findings from Discovery

**nmrglue 2D API:**
- `ng.bruker.read_pdata()` works for 2D, returns (dic, data) where data.shape = (F1, F2)
- F1 nucleus from `dic['acqu2s']['NUC1']`
- F2 nucleus from `dic['acqus']['NUC1']`
- `ng.bruker.guess_udic()` returns universal dict with labels per dimension
- `ng.fileiobase.uc_from_udic(udic, dim=N)` gives unit converter for each dimension

**Experiment Detection:**
- 2D experiments have `acqu2s` file (F1 acquisition parameters)
- Pulse program names indicate experiment type (hsqc*, hmbc*, cosy*, etc.)
- Homonuclear: F1 nucleus == F2 nucleus (COSY, TOCSY, NOESY)
- Heteronuclear: F1 nucleus != F2 nucleus (HSQC, HMBC)

## Tasks

### Task 1: Add read_2d method to BrukerReader

Add to `src/lucy_ng/readers/bruker.py`:

```python
@staticmethod
def read_2d(experiment_dir: str | Path) -> Spectrum2D:
    """Read a Bruker 2D experiment directory and return Spectrum2D."""
```

Implementation:
- Use `ng.bruker.read_pdata()` to read processed 2D data from pdata/1/
- Read both acqus and acqu2s for F1/F2 parameters
- Extract F2 nucleus from `dic['acqus']['NUC1']`
- Extract F1 nucleus from `dic['acqu2s']['NUC1']`
- Generate ppm scales using `ng.fileiobase.uc_from_udic()` for each dimension
- Detect experiment type from pulse program name
- Return populated `Spectrum2D` instance

**Acceptance**: Function reads Bruker 2D directory and returns Spectrum2D.

### Task 2: Implement experiment type detection

Add helper function to detect 2D experiment type from pulse program:

```python
def _detect_experiment_type(pulse_program: str, f1_nucleus: str, f2_nucleus: str) -> str:
    """Detect 2D NMR experiment type from pulse program and nuclei."""
```

Detection logic:
1. First try pulse program pattern matching:
   - Contains "hsqc" or "inv4" without "hmbc" → HSQC
   - Contains "hmbc" → HMBC
   - Contains "cosy" → COSY
   - Contains "tocsy" or "mlev" or "dipsi" → TOCSY
   - Contains "noesy" → NOESY
   - Contains "roesy" → ROESY
2. Fallback: if F1 == F2 nucleus → unknown homonuclear, else unknown heteronuclear
3. Raise ValueError if cannot determine type

**Acceptance**: Correctly identifies HSQC, HMBC, COSY, NOESY from test data.

### Task 3: Handle parameter extraction for 2D

Add helper to get parameters from both acqus and acqu2s:

```python
def _get_2d_params(dic: dict) -> dict:
    """Extract parameters for both dimensions of a 2D experiment."""
```

Extract:
- F2 nucleus, frequency, sweep width from acqus
- F1 nucleus, frequency, sweep width from acqu2s
- Solvent (from acqus)
- Pulse program (from acqus)
- Number of scans (from acqus)

**Acceptance**: Parameters extracted correctly for HSQC, HMBC, and COSY experiments.
